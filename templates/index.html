About Nmap

Nmap (Network Mapper) is a free and open-source tool used for:

  • Network discovery and security auditing
  • Host and service detection across networks
  • Vulnerability scanning and penetration testing
  • Network inventory and asset management
  • Monitoring host or service uptime

Created by Gordon Lyon (known by his pseudonym Fyodor) in 1997, Nmap has become the de facto standard for network exploration and security auditing. It's widely used by:

  • System and network administrators
  • Penetration testers and ethical hackers
  • Cybersecurity professionals
  • IT auditors and compliance officers

Technical Capabilities

Nmap uses raw IP packets to determine:

  • Available hosts on the network
  • Services (application name and version)
  • Operating systems and versions
  • Packet filters/firewalls in use

Advanced Features

The Nmap Scripting Engine (NSE) allows for:

  • Network discovery
  • More sophisticated version detection
  • Vulnerability detection
  • Backdoor detection
